创建数据库表的基本语法格式：

```
create table <table_name>(
		C1 D1,
		C2 D2,
		...,
		An Dn,
		<完整性约束1>，
		...
		<完整性约束k>,
);
```

Ci是列名，Di是该列的域，Di指定了Ci的类型和可选值的约束。

完整性约束包括：

1. primary key(Aj1, Aj2, ...):主码，唯一主键。关系集合中， 没有任何两行在所有的主键取值上完全相同。

2. foreign key(Aj1, Aj2,...)

3. not null

   
修改数据库表结构:

```
alter table <table_name> add A,D;
alter table <tn> drop A;
```
